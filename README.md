# Tyrese Hawthorne Portfolio

Static portfolio website for `t-hawthorne.com`.

## Live Site Folder

The deployable site is in:

```txt
portfolio-site/
```

Cloudflare Pages settings:

- Framework preset: None / Static HTML
- Build command: leave blank
- Output directory: `portfolio-site`

## Editing With Codex

For simple updates, ask Codex to edit files in `portfolio-site/`.

Examples:

```txt
Update my email address on the website.
Add a new project card for security audit reports.
Create an articles section with individual article pages.
```

After changes are committed and pushed to GitHub, Cloudflare Pages can redeploy automatically from this repository.

## Domain

Domain setup notes are in:

```txt
portfolio-site/DOMAIN_SETUP.md
```
